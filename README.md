# Lombok
## É uma ferramenta Open-Source que remove toda a codificação padrão conhecida numa classe Java.
### Anotações que diminuem consideravelmente o tamanho do código:
#### @Getter
#### @Setter
#### @NoArgsConstructor
#### @AllArgsConstructor
#### @EqualsAndHashCode
#### @ToString
